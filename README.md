# Byte Storage Solutions – Web & Network Docs

Ez a repository tartalmazza a Byte Storage Solutions hivatalos weboldalának és 
a cég hálózati dokumentációjának forráskódját. A projekt egyszerű, reszponzív 
frontendre épül, és bemutatja a cég három fő részlegét: Raktár, Iroda, Szerviz.

## Tartalom
- Weboldal forráskód (HTML/CSS/JS)
- Cégstruktúra összefoglaló
- Hálózati felépítés (VLAN-ok, szegmensek, Packet Tracer modell)

## Felépítés
- `/assets` – képek, CSS, JS
- `/warehouse`, `/office`, `/service` – részlegek oldala
- `index.html` – főoldal
